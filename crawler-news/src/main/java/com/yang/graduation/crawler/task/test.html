韩国瑜被罢免 赖清德：不再有百年难得的政治奇才

2020年06月07日 15:14:32
来源：海峡导报

台湾地区副领导人赖清德今天表示，高雄人的民主实践，让大家深刻反省，主权在民的社会里，不再有百年难得的政治奇才。

高雄市长韩国瑜罢免案过关，成为岛内首位被罢免的“直辖市长”。

曾夸赞韩国瑜是难得一见“政治奇才”的赖清德，晚间在脸书发文表示，高雄人的民主实践，让大家深刻反省，主权在民的社会里，不再有百年难得的政治奇才，只有恒久不变的公民力量。

他称，政府的权力来自于人民，在台湾，不会是一句空虚无力的口号。而今天，高雄人用强烈意志，展现了这样的理想和价值。“民主无价、高雄加油。”

另据报道，韩国瑜罢免投票今天通过，新北市发言人蒋志薇转述新北市长侯友宜说法表示，“民意已经做了决定，虽然遗憾，我们尊重。”

侯友宜说，韩市长的团队这段时间辛苦了，从市政到防疫工作，他们都很认真努力，不要气馁，未来如果再有为社会服务的机会，我们一起加油！

综合台湾媒体报道
